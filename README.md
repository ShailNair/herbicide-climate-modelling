This repository contains three main analysis scripts:

1. **herbicide_erosivity_model.R**: Project future coastal herbicide discharge under climate change using XGBoost machine learning framework.
   
2. **herbicide_sst_phytoplankton_gam.R**: Quantify herbicide-SST interaction effects on marine phytoplankton functional groups using Generalized Additive Models (GAMs).
  
3. **phytoplankton_herbicide_meta_analysis.R**: Meta-analysis of experimental phytoplankton responses to herbicide exposure at environmentally relevant concentrations.
